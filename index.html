¿Por qué este sitio web y no WIX?

Este sitio web nace de la necesidad de realizar la Actividad 4 - Mi sitio Web con HTML y CSS, a pesar de que la actividad indica realizarlo a través de un constructor de sitios web, considerando el proceso académico que estamos llevando a cabo, opté por realizar este ejercicios con HTML, CSS, JS y Bootstrap. De esta manera garantizo los requisitos asociados a la actividad, además de que hago una actividad más acorde a nuestra labor como ingenieros de software.

Este sitio estará publicado a través de github pages, donde también comparto el link del repositorio en caso tal de que desee revisarlo. De antemano muchas gracias y espero no generarle inconvenientes con la opción tomada.
